# mysql-awslinux
Connect mysql to linux 
Coomands used:
  1  yum update -y
    2  yum install httpd -y
    3  httpd -v
    4  systemctl enable httpd
    5  systemctl start httpd
    6  curl ifconfig.me
    7  yum install mysql
    8 mysql -h<database end pointlink> -u <your database username> -p
